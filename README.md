# Password Strength Checker

A simple command-line tool that rates how strong a password is and
explains how to make it better.

## What it checks
- Length (8+ minimum, 12+ recommended)
- Mix of uppercase, lowercase, numbers, and special characters
- Whether the password is on a list of commonly-used weak passwords

## How to run
1. Make sure Python 3 is installed.
2. Run the script:
   python3 Password_Checker.py
3. Enter a password when prompted to see its strength score and tips.

## Note
Built for educational purposes as part of learning security fundamentals.
